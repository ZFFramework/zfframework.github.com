convert integer number (int, long, etc) to string

integer number can be signed or unsigned, whatever your compiler support
typical usage:

zfstring s = ...;
zfint n = ...;
// append result to s and return it as result
zfsFromIntT(s, n);
// return a new zfstring as result
s = zfsFromInt(n);
