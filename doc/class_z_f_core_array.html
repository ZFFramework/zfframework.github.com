Detailed Description

template<typename T_Element>
class ZFCoreArray< T_Element >

light weight array

Warning
element must support: constructor with no argument, copy constructor to create from existing value, operator = to copy value

REMARKS:

  • ZFCoreArray is designed to reduce dependency of STL or other third-party containers, aimed for light weight
  • use other containers such as ZFArray in your app, if performance is really a matter, choose any other containers you like, but try not to export them
