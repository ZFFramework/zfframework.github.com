Detailed Description

data used by ZFIO when finding files

typical usage:

ZFIOImpl *io = ZFIOImplForPathType(pathType);
ZFIOFindData fd;
if(io->ioFindFirst(fd, path)) {
do {
// do something
} while(io->ioFindNext(fd));
io->ioFindClose(fd);
}
