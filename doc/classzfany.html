Detailed Description

util method to cast ZFObject types freely

usage:

YourObjectType *obj = ...;
zfany any = obj;
// cast to any other object type, return null if not able to cast
AnotherObjectType *another = any;
// or use toObject
ZFObject *orig = any.toObject();
