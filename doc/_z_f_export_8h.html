the variable are exported as ZFMETHOD_FUNC_DECLARE_0
usage:

// in header file
ZFEXPORT_VAR_DECLARE(ZFLIB_APP, YourType, yourName)
// in source file
ZFEXPORT_VAR_DEFINE(YourType, yourName, yourInitValue)
// access
YourType &v = yourName();
// modify
yourName(v);
