safely cast ZFObject types, return null if not desired type

types can only be ZFObject or ZFInterface types

ADVANCED:

  • cast between ZFInterface or it's children types may take much time for dynamic checking, use with caution
  • try not to cast from and to the base ZFInterface * type

cast ZFObject types without safe check, use only for performance

ADVANCED:

  • this method would have better performance while cast ZFObject types to ZFObject types, while cast ZFInterface types would always consume some CPU for dynamic check
  • try not to cast from and to the base ZFInterface * type
