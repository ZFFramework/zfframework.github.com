◆ ZFLuaState()

void*() ZFLuaState ( )

global native lua state currently used by ZFLuaExecute, ensured not null

ZFFramework can be used in any native lua state (by using ZFLuaExecute), but only if you have attached them
by default ZFLuaState would have a builtin state that already setup properly, while you can attach your own native lua state to ZFFramework, by ZFLuaStateAttach/ZFLuaStateDetach
the builtin lua state can also be changed by ZFLuaStateChange
