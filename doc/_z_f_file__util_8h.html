◆ ZFFilePathFormat()

zfbool() ZFFilePathFormat ( zfstring &  ret,
const zfchar *  src,
zfindex  srcLen = (((zfindex) -1)) 
)

format path according ZFFileSeparator

format rules:

  • ensured no '/' at tail
  • all extra '/' and '\' would be replaced with single '/'
  • strip spaces and tabs at head and tail
  • replace these escape chars:
    • \" to double quote
    • \ to space
  • remove "./" at head
  • remove "/." at tail
  • remove "/./" at anywhere
  • null src is treated as error, while empty string src is not an error
  • single "/" src would result empty string
