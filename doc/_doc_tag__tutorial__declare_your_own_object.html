Declare your own object

to use ZFFramework's object, you must follow some rules:

zfclass MyObject : zfextend ParentObject {
ZFOBJECT_DECLARE(MyObject, ParentObject)
};
// in cpp files
ZFOBJECT_REGISTER(MyObject)
