◆ ZFFileFileFindFirst()

zfbool() ZFFileFileFindFirst ( ZFFileFindData &  fd,
const zfchar *  path 
)

find file or directory, similar to FindFirstFile under Windows

path supports path only, without wildcard support, e.g. "/path/" or "/path", "." and ".." won't be included
typical usage:

ZFFileFindData fd;
if(ZFFileFileFindFirst(fd, path))
{
do
{
// do something
} while(ZFFileFileFindNext(fd));
ZFFileFileFindClose(fd);
}
Returns
false if error or nothing found
Warning
strings in ZFFileFindData would be deleted after each findXXX call, you must save it if need future use
See also
ZFFileFindData, ZFFileFileFindFirst, ZFFileFileFindNext, ZFFileFileFindClose
Note
path must be well formed, use ZFFilePathFormat if necessary

◆ ZFFileFileRead()

zfindex() ZFFileFileRead ( ZFToken  token,
void *  buf,
zfindex  maxByteSize 
)

read file

return size read, even if error occurred (whether error occurred or eof, could be accessed by ZFFileFileIsError and ZFFileFileIsEof)
typical usage:

zfindex sizeRead = 0;
while((sizeRead = ZFFileFileRead(token, buf)) > 0
&& !ZFFileFileIsError(token))
{
// do something with buf
}
if(ZFFileFileIsError(token))
{
// exit because of error occurred, read fail
}
else if(ZFFileFileIsEof(token))
{
// exit because of eof, read success
}

note that the tail '\0' won't be written to buf

Warning
ZFFile always read files in binary mode, for text files, you must make sure the file is UTF8 encoded without BOM

◆ ZFFileFileWrite()

zfindex() ZFFileFileWrite ( ZFToken  token,
const void *  src,
zfindex  maxByteSize = (((zfindex) -1)) 
)

write file, see ZFFileFileRead

return size written, even if error occurred (whether error occurred or not, could be accessed by ZFFileFileIsError)
typical usage:

zfindex sizeWritten = ZFFileFileWrite(token, src, size);
if(ZFFileFileIsError(token))
{
// write fail
}
if(sizeWritten != size)
{ // you can also check by sizeWritten
// write fail
}


maxByteSize could be zfindexMax(), which means ZFFileFileWrite should be stopped when reached 0x00 in src, usually to output a UTF8 string
