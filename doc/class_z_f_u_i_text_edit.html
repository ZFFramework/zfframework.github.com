Detailed Description

view to supply simple edit logic

Note
this view is used for simple text editing only, for simple text display, use ZFUITextView, for rich text edit, you should supply your own implementation
this view supply quite simple text edit logic, designed to be lightweighted and able to be embeded to other complex views
