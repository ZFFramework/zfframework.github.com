◆ sdlMouseGrabCallback

zfbool(* ZFImpl_sys_SDL_View::sdlMouseGrabCallback) (ZFImpl_sys_SDL_View *owner, SDL_Event *sdlEvent, zfint x, zfint y)

extra callback for impl to achieve custom mouse event dispatch logic

the dispatch logic:

  1. during mouseTest, called with null sdlEvent for each sdl view, to check whether the view want to grab mouse event
  2. if a sdl view grabbed mouse down event, all later events would be sent to that view's sdlMouseGrabCallback directly, until next mouse down event
