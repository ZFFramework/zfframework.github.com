Detailed Description

util to hold and connect ZFInput and ZFOutput

usage:

void outputFunc(ZF_IN const ZFOutput &callback) {...}
void inputFunc(ZF_IN const ZFInput &callback) {...}
zfobj<ZFIOBufferByXxx> io;
outputFunc(io->output()); // output data to io's internal buffer
inputFunc(io->input()); // input data from io's internal buffer
ZF_IN
#define ZF_IN
dummy macro that shows the param used as required input

once created, the ZFIOBuffer would be retained by the returned ZFInput/ZFOutput by ZFCallbackTagKeyword_ZFIOBuffer, it's ensured safe to release the owner ZFIOBuffer's reference
