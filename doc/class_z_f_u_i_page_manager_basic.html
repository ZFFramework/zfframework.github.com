Detailed Description

basic page manager

minimum steps to use the basic page manager:

  1. during ZFUIPageManager::managerOnCreate: (either by override or by event observer)
    • create your manager's view tree
    • setup ZFUIPageManagerBasic::pageContainer
    • prepare your home page, and create it by ZFUIPageManager::requestPageCreate
  2. call embededCreateWindow to create window and attach
  3. finally, show the window

◆ managerContainer()

virtual ZFUILinearLayout* const& ZFUIPageManagerBasic::managerContainer ( )
inlinevirtual

the manager container as view

by default, a manager use managerContainer as view to hold the page manage logic, you should manually add the container to view tree, and manually manage the manager's life cycle by ZFUIPageManager::embededCreate series, this was designed for most compatibility and flexibility

for most case, it's common to use ZFUIWindow to hold the manager, so we also have an util method embededCreateWindow, which would:

  • create a ZFUIWindow
  • attach managerContainer to the window
  • call embededCreate
  • when the window shows, call embededResume
  • when the window hides, call embededPause
  • when the window destroy, call embededDestroy
  • when manager destroy, detach from the window, then hide the window
